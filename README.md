# ABM Academy - EEE Communication Interactive E-Book & Simulator

An interactive, premium, single-file HTML5 learning platform designed for EEE job preparation and academic learning in Communication Engineering.

## 📖 Overview

This platform converts traditional textbook and exam-based mathematical problems into an interactive e-book experience. It features dynamic simulators, concept quizzes, and a split-card UI that displays solved steps alongside EEE exam formulas.

### Key Categories & Chapters:
1. **Sampling & Nyquist Rate**: Calculations for composite signals, multiplied signals, and sampling overheads.
2. **PCM Bit Rate & TDM**: Calculations for TDM frame lengths, frame durations, synchronization/control bit rates, and transmission bandwidths.
3. **SQNR & Companding**: Analysis of quantization levels (strictly rounded to $2^n$), step sizes, uniform SQNR, and $\mu$-law companding SQNR.
4. **Storage & Dynamic Range**: Stereophonic and monophonic music storage requirements, bit rate conversions, and dynamic range to quantization bit calculations.

---

## ⚡ Features

- **Split-Card Layout**: Problem statements and detailed derivations on the left, with concise Bengali formula guides on the right.
- **Live Interactive Simulators**:
  - Automatically fill inputs with preset parameters from solved textbook problems.
  - Smooth-scroll and visual glow indicators for active widgets.
  - Dynamically updates output formulas and displays recalculated values in real-time.
- **Dual Theme Switch**: Seamlessly transition between Dark Mode and an eye-friendly, soft blue-tinted Light Mode (`#f0f5fa`) designed to reduce eye strain.
- **Concept Quizzes**: Interactive multiple-choice questions with real-time feedback and explanation cards.
- **MathJax Integration**: Dynamic LaTeX equations compilation directly inside the browser.
- **Zero AI Branding**: Fully white-labeled.

---

## 🚀 How to Run

Since the application is built entirely as a single self-contained HTML file:
1. Download or clone this repository.
2. Double-click [index.html](file:///./index.html) or open it in any modern web browser (Google Chrome, Microsoft Edge, Mozilla Firefox, Safari, etc.).
3. No local servers or installations are required!
